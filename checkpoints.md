# Checkpoints

Project checkpoint history has been split into the package directories.

- [zig_btree/checkpoints.md](zig_btree/checkpoints.md): B-tree goals and performance work.
- [zig_phmap/checkpoints.md](zig_phmap/checkpoints.md): parallel-hashmap goals and performance work.

Packages without a checkpoint file do not currently have recorded checkpoint history.
